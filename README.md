# Requirements

Rename firebase-back.js to firebase.js and fill the below keys with your values from firebase:

```json
const firebaseConfig = {
  apiKey: "",
  authDomain: "",
  projectId: "",
  storageBucket: "",
  messagingSenderId: "",
  appId: "",
  measurementId: ""
};
```

Create an admin user in firebase

Log in with the user

# Start the application

npm start
